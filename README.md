# 🌧️ Kerala Rainfall Analysis (1901–2018)

Exploratory analysis of Kerala’s rainfall patterns (1901–2018) with a focus on **extreme monsoon events** and **flood years**.

This project serves as an entry point into **GeoAI / Environmental Intelligence** — combining data science with climate and geospatial analysis.

---

## 🎯 Project Objectives

- Analyze **seasonal and annual rainfall trends**
  - JJAS (Southwest Monsoon)
  - OND (Retreating Monsoon)
  - Annual totals  
- Identify **extreme rainfall years** using statistical thresholds (95th percentile)  
- Compare **flood years vs non-flood years** in terms of rainfall intensity  
- Study **decadal monsoon trends** to understand long-term changes  

---

## 📂 Dataset

### 1. `Kerala-Rainfall-Historical.csv`
- Source: *IMD / Climate Data Portal*  
- Contains:
  - Monthly rainfall data  
  - Seasonal aggregates (JJAS, OND, ANNUAL)  

### 2. `kerala.csv`
- Contains:
  - Annual rainfall data  
  - Flood year labels (**YES / NO**)  

---

## 🛠️ Tools & Libraries

- **Python**
- **Pandas** – Data cleaning & analysis  
- **Matplotlib / Seaborn** – Data visualization  
- **Jupyter Notebook** – Interactive exploration  

---

## 🔍 Key Insights

- Kerala experiences **high variability** in monsoon rainfall  
- Extreme rainfall years (e.g., **2018 flood year**) exceed the **95th percentile threshold** significantly  
- **Retreating monsoon (OND)** often contributes disproportionately to flood-related years  
- **Decadal analysis** indicates changing monsoon intensity over time
